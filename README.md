## 功能特点

### 图表类型
- **折线图**: 温度变化趋势分析
- **柱状图**: 淘宝天猫GMV数据展示
- **水平柱状图**: 线上替代率对比
- **堆积图**: 物流成本统计
- **直方图**: 人脸识别数据分布
- **饼图/环形图**: 数据占比分析（可切换）
- **散点图**: 数据相关性分析
- **箱线图**: 数据分布统计
- **雷达图**: 多维度数据对比
- **误差棒图**: 实验数据测量结果

### 数据接口功能
- **手动输入**: 通过表单直接输入数据
- **Excel导入**: 支持上传Excel文件自动导入数据
- **实时更新**: 数据变化即时反映在图表上
- **数据验证**: 自动检查输入数据的格式和完整性

### 其他功能
- **图表导出**: 支持将图表导出为PNG图片
- **响应式设计**: 适配不同屏幕尺寸
- **交互式操作**: 标签页切换，动态数据刷新
- **美观界面**: 现代化UI设计，良好的用户体验

## 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **图表库**: Apache ECharts 5.4.3
- **Excel处理**: SheetJS (XLSX.js)
- **部署**: 可部署在任何静态文件服务器

## 快速开始

### 本地运行
1. 克隆或下载此项目
2. 直接用浏览器打开 `完整动态数据可视化平台.html` 文件
3. 或者使用本地服务器运行：
   ```bash
   # 使用Python
   python -m http.server 8080
   
   # 使用Node.js
   npx serve -p 8080
   
   # 使用PHP
   php -S localhost:8080
   ```
4. 访问 `http://localhost:8080/完整动态数据可视化平台.html`

## 使用说明

### 查看图表
1. 点击顶部标签切换不同图表类型
2. 每个图表页面包含图表展示区域和数据输入区域
3. 鼠标悬停在图表上可查看详细数据

### 更新图表数据
#### 手动输入
1. 在数据输入区域填写相应数据
2. 点击"提交数据"按钮
3. 图表将自动更新显示新数据

#### Excel文件导入
1. 点击"选择Excel文件"按钮
2. 选择符合格式要求的Excel文件
3. 系统自动解析并更新图表数据

### 导出图表
1. 点击图表右上角的"导出图表"按钮
2. 图表将以PNG格式下载到本地

### 刷新数据
1. 点击图表右上角的"刷新数据"按钮
2. 系统将生成随机测试数据并更新图表

## Excel文件格式要求

每种图表类型有不同的Excel文件格式要求：

### 折线图
- 第一列：日期
- 第二列：最高温度
- 第三列：最低温度

### 柱状图
- 第一列：年份（如FY2013）
- 第二列：GMV值

### 水平柱状图
- 第一列：类别名称
- 第二列：替代率值

### 堆积图
- 第一列：月份（1-12）
- 第二列：公司A费用
- 第三列：公司B费用
- 第四列：公司C费用

### 直方图
- 第一列：置信度区间（如0-10）
- 第二列：频次

### 饼图/环形图
- 第一列：来源/类别名称
- 第二列：数值

### 散点图
- 第一列：X坐标值
- 第二列：Y坐标值

### 箱线图
- 第一列：2017年数据
- 第二列：2018年数据

### 雷达图
- 第一列：个人兴趣值
- 第二列：平均兴趣值

### 误差棒图
- 第一列：实验条件
- 第二列：测量值
- 第三列：误差值
