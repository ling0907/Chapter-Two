# 更新日志

本文档记录了动态数据可视化平台的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2023-11-07

### 新增
- 初始版本的动态数据可视化平台
- 包含11种不同类型的图表：
  - 折线图：温度变化趋势分析
  - 柱状图：淘宝天猫GMV数据展示
  - 水平柱状图：线上替代率对比
  - 堆积图：物流成本统计
  - 直方图：人脸识别数据分布
  - 饼图/环形图：数据占比分析（可切换）
  - 散点图：数据相关性分析
  - 箱线图：数据分布统计
  - 雷达图：多维度数据对比
  - 误差棒图：实验数据测量结果

### 功能
- 手动数据输入功能
- 图表导出为PNG图片
- 响应式设计，适配不同屏幕
- 标签页切换功能
- 数据刷新功能，生成随机测试数据

### 技术实现
- 使用Apache ECharts 5.4.3进行图表渲染
- 使用HTML5、CSS3和原生JavaScript实现
- 采用现代化UI设计

## [1.1.0] - 2023-11-07

### 改进
